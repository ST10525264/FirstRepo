The app being coded today is used to determine items packed for a campsite
The Splash Screen is the homepage with displays the app's logo as well as the name of the app
The splash Screen will transition to the Main screen after 3 seconds
The Main screen consists of a button "Add Gear", a display for the total items packed as well as has a dark mode display
The Detailed View screen uses a display the categorized items in a table form
The Detailed View screen also has a button "Back to Base" which takes the user back to the Main screen
The comments are used to provide further details or a description of the items

The Code
The code uses a while loop with an if statement to calculate the items packed
Each button is coded for specific use
The "Add Gear" button is coded to return the items when clicked
The "Back to Base" button is coded to return the user to the Main screen
The arrays are used to store the item details effectively 
The sample data set was used to for the exam booklet which includes a tent, marshmellows and a flashlight
